# Phase 02 Visual Design System Baseline

## Layout
- Desktop-first dashboard shell with fixed left navigation and top header.
- Main content uses card-based modular sections and table-heavy panels.
- Header includes:
  - Search/filter utility zone
  - `Settings` icon
  - `Scenario Lab` icon
  - Language switch `TR` / `EN`

## Core Components
- KPI card
- Data table with row actions
- Status badge
- Button variants (primary, secondary, danger, ghost)
- Modal and confirmation dialog
- Drawer panel
- Tabs (`Pending` / `Processed`)
- Segmented mode control (`Manual` / `Auto-Sim`)

## State Conventions
- Default
- Loading
- Empty
- Success
- Warning
- Error

## Accessibility Baseline
- Semantic button roles for all interactive controls.
- Visible focus states for keyboard users.
- Color + text dual coding for critical statuses.
