# Phase 02 Theme Token Sheet

## Typography
- `font-family.base`: Tailwind `font-sans`
- `font-weight.heading`: `600` / `700`
- `font-weight.body`: `400` / `500`

## Color Tokens
| Token | Tailwind Value | Usage |
| --- | --- | --- |
| `color.bg.page` | `gray-50` | App background |
| `color.bg.surface` | `white` | Cards, panels |
| `color.border.soft` | `gray-100` | Soft separators |
| `color.border.default` | `gray-200` | Standard borders |
| `color.text.primary` | `gray-800` | Main text |
| `color.text.secondary` | `gray-600` | Secondary text |
| `color.text.muted` | `gray-400` | Helper/meta |
| `color.primary.strong` | `indigo-700` | Top nav, emphasis |
| `color.primary.default` | `indigo-600` | Primary CTA |
| `color.primary.soft` | `indigo-100` | Secondary CTA bg |
| `color.primary.tint` | `indigo-50` | Surface tint |
| `color.primary.active` | `indigo-200` | Active nav/hover |
| `color.status.danger` | `red-600` | High-risk state |
| `color.status.warning` | `yellow-600` | Margin warning |
| `color.status.success` | `green-600` | Competitive/opportunity |
| `color.status.info` | `blue-500` | Informational accent |

## Spacing
- Base spacing unit: `4px`.
- Card paddings: `16px` / `24px`.
- Grid gap: `16px` / `24px`.

## Radius And Elevation
- Radius:
  - `card`: `12px`
  - `button`: `8px`
- Elevation:
  - Base card shadow: subtle (`shadow-sm` equivalent)
