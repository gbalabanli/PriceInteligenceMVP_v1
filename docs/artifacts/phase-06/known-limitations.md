# Known Limitations

## Current MVP Constraints
- All data is mock and held in browser memory/localStorage.
- No backend API, authentication, or persistent audit datastore exists.
- Dynamic pricing is deterministic simulation, not production automation.
- A/B snapshot export is JSON-only (no CSV/XLSX export in this iteration).
- Chart visualizations are minimal; the dashboard is table/KPI driven.
- Accessibility is baseline-level and should be hardened in next iteration.

## Non-Goals In This Cycle
- Real competitor data ingestion/scraping pipeline.
- Real-time collaborative usage and multi-user conflict handling.
- Production deployment hardening and observability stack.
